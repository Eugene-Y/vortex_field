*In Albert's GR there is no rotation -*  
*So drag here to play and get inspiration*  

Every pair of fluid cells has an instantaneous center of rotation. This is an interactive visualizer of the accumulated instantaneous rotation.

The "pair range" slider controls which pairs count: slide right to include nearby cells, left for distant ones. At -1 and 1 all the pairs count. NB: this is very GPU intense.

Sometimes it creates peculiar patterns resembling [spiral galaxies](https://eugene-y.github.io/vortex_field/?gridSize=100&dampingLoss=1.000e-7&simSpeed=1.32&brushRadius=2.00&brushStrength=209&patternScale=0.370&pairRange=0.128&boundary=1&vorticity=0.00&pressureIters=40&pattern=disk-spin&velBrightness=50&rotBrightness=82), [submarine creatures](https://eugene-y.github.io/vortex_field/?gridSize=100&dampingLoss=1.000e-7&simSpeed=4.57&brushRadius=2.00&brushStrength=366&patternScale=0.370&pairRange=0.0579&boundary=1&vorticity=0.00&pressureIters=40&pattern=trilines&velBrightness=50&rotBrightness=100), [magnetic poles](https://eugene-y.github.io/vortex_field/?gridSize=100&dampingLoss=5.372e-8&simSpeed=0.166&brushRadius=1.21&brushStrength=500&patternScale=0.140&pairRange=0.0493&boundary=0&vorticity=0.00&pressureIters=90&pattern=gridlines&velBrightness=52&rotBrightness=100) or [plasma dynamics](https://eugene-y.github.io/vortex_field/?gridSize=100&dampingLoss=1.000e-8&simSpeed=4.17&brushRadius=0.697&brushStrength=500&patternScale=0.470&pairRange=0.0345&boundary=0&vorticity=0.00&pressureIters=57&pattern=decagon&velBrightness=75&rotBrightness=100).
